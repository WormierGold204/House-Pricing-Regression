Information about the App

This application is designed to predict house prices based on various features of the houses. It provides functionalities for training regression models, making predictions, and explore data analysis.

Features:

  • Training: it is possible to train a model by giving it a name, choosing which type of regression to use and selecting the features on which the training will be based (the features names are the same as the columns of the dataset).
  • Prediction: it is possible to make a prediction by selecting a model (between those that have been trained) and providing the values for all the features requested.
  • Models: it is possible to view the models that have been trained, and also delete them if desired.
  • EDA: it is possible to view performed exploratory data analysis on the dataset.
